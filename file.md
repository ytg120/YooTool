# **GMaker**
Game Maker

A python tkinter game maker.

## Installation
1. Click [here](https://github.com/ytg120/GMaker/releases/latest)
2. Click assets and install the .zip file(windows 64bit supported).
3. Extract the .zip file
4. Run the main.exe file.

# **GMaker**
게임메이커

파이썬 Tkinter 게임 제작 툴.

## 설치
1. [여기](https://github.com/ytg120/GMaker/releases/latest)를 클릭합니다.
2. Assets을 클릭해서 .zip파일을 설치합니다(윈도우 64비트 지원됨.)
3. .zip 파일 압축을 해제합니다.
4. main.exe 파일을 실행합니다.
