# **GMaker**
Game Maker

A python tkinter game maker.

## Installation
1. Click [here](https://github.com/ytg120/GMaker/releases/latest) to install(windows 64bit supported).
2. Extract the .zip file
3. Run the main.exe file.

# **GMaker**
게임메이커

파이썬 Tkinter 게임 제작 툴.

## 설치
1. [여기](https://github.com/ytg120/GMaker/releases/latest) 클릭하여 설치(윈도우 64비트 지원됨.)
2. .zip 파일 풀기.
3. main.exe 파일을 실행.
